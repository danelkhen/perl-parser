# perl-parser
Perl parser written in TypeScript

## Installation
```
npm install -g typescript typings
git clone perl-parser
cd perl-parser
npm install
```

## Compilation
```
tsc -p src-node
tsc -p test
```

* add `-w` to watch and recompile automatically

## Running the server
Windows
```
node built\src-node\src-node\server.js
```
Linux
```
node built/src-node/src-node/server.js
```
